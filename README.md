## User documentation

### Production build

```shell
npm install
npm run build
```

## Developer documentation

### Start hacking

```shell
npm install
npm run dev
```

### Lint

You should already have the "fix lint on commit" feature available.

If you still want to lint your code manually:

```shell
npm run lint:fix
```

### Test

```shell
npm run test
```
